# Claim Matching using Multilingual MPNet

## Objective
This task focuses on measuring semantic similarity between pairs of claims.

## Dataset
- Claim pairs
- Four similarity classes:
  - Very Similar
  - Somewhat Similar
  - Somewhat Dissimilar
  - Very Dissimilar

## Model Used
- Multilingual MPNet

## Methodology
1. Data preprocessing
2. Pairwise text encoding
3. Semantic similarity classification
4. Class balancing
5. Cross-validation and performance evaluation

## Technologies
- Python
- Google Colab
- Sentence Transformers
- PyTorch

## Learning Outcomes
- Semantic similarity analysis
- Cross-validation techniques
- Handling class imbalance
- Multilingual sentence embeddings
