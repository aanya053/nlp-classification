# nlp-classification
Multilingual NLP classification project with three subtasks, including model development, evaluation, and performance optimization.

## Overview

The project consists of three NLP subtasks:

### Subtask 1A: English Claim Detection
- Binary classification (Claim / Not Claim)
- Model: BERT

### Subtask 1B: Indian Language Claim Detection
- Multiclass classification
- Model: XLM-RoBERTa

### Subtask 1C: Claim Matching
- Semantic similarity classification
- Model: Multilingual MPNet
