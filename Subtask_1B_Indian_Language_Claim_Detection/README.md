# Indian Language Claim Detection using XLM-RoBERTa

## Objective
This task focuses on detecting claims in multilingual Indian-language texts.

## Dataset
- Multilingual Indian language dataset
- Three classes:
  - Yes
  - No
  - Probably

## Model Used
- XLM-RoBERTa

## Methodology
1. Data cleaning and preparation
2. Label encoding
3. Tokenization using XLM-R tokenizer
4. Fine-tuning for multiclass classification
5. Evaluation using Precision, Recall, F1-Score and ROC-AUC

## Technologies
- Python
- Google Colab
- Hugging Face Transformers
- PyTorch

## Learning Outcomes
- Multilingual NLP
- Transformer-based classification
- Handling low-resource languages
- Performance evaluation metrics
